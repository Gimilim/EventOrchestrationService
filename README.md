# EventOrchestrationService
Данный проект реализует апи для работы с событиями.
Используется актуальаня версия .net (10) на момент реализа3ции проекта

Структура проекта:

/models -- модели проекта

/controllers -- контроллеры проекта

Подключенные технологии:
-Используется fluentValidation (в основном "на вырост")
-Есть планы на переход на OpenApi вместо сваггера

Полезное:
`dotnet restore` -- для подтягивания зависимостей
`dotnet build` -- сборка
`dotnet run` -- запуск

Условности:
БД пока не подключена, вместо этого данные хранятся в памяти (будет изменено в будущем)
