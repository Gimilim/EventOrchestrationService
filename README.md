# EventOrchestrationService
Данный проект реализует API для работы с событиями.
Используется актуальная версия .net (10) на момент реализации проекта

Структура проекта:

/Models -- модели проекта
/Controllers -- контроллеры проекта
/Data -- хранение данных и настройки их хранения

Подключенные технологии:
-Используется fluentValidation (в основном "на вырост")
-Подключены Moq и xUnit для тестов
-Есть планы на переход на OpenApi вместо сваггера
-Пока что используется Sqlite для работы с данными
-Есть инициализатор тестовых данных (файл DbInitializer)

Общая информация:
-Реализован глобальный перехватчик ошибок для всех исключений
-Реализована пагинация для запроса всех событий
-Методы базово покрыты тестами

Полезное:
`dotnet restore` -- для подтягивания зависимостей
`dotnet build` -- сборка
`dotnet run` -- запуск проекта
`dotnet test` -- запуск тестов

Условности:
-БД пока не подключена, вместо этого данные хранятся в памяти (будет изменено в будущем)
-Тесты написаны не оптимально. Рефактор планируется после введения dbcontext'а
